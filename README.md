## ğŸ¨ **CUSTOMIZATION GUIDE**

### **Colors**
Update `tailwind.config.js` brand palette:
```javascript
extend: {
  colors: {
    'primary-red': '#FF0000',
    'dark-red': '#CC0000',
    'light-red': '#FF6666',
    'accent-red': '#FF3333',
    'blood-red': '#8B0000',
  }

```

Modify `globals.css` for custom animations:
- Cursor effects
- Scroll animations
- Hover transitions
- Loading sequences

Update component files with your information:
- Personal details in `Hero.tsx`
- Skills and experience in `About.tsx`
- Work history in `Experience.tsx`
- Projects in `Projects.tsx`
