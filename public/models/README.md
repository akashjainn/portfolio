# 3D Model Assets

This directory contains 3D models for interactive portfolio showcases.

## Files

- `vine-trees.fbx` - Custom-created vine and tree models for the VR Bomb Defusal Puzzle game (1.1MB)

## Usage

Models are loaded using Three.js FBXLoader in the ModelViewer component. They can be viewed interactively with:
- Orbit controls (rotate, pan, zoom)
- Dynamic lighting
- Environment mapping
- Contact shadows
